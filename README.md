# Detecting-IFS

This repository contains code written for investigations into the detection and separation of IFS from data. This research was part of an MSci thesis in Mathematics.

## Usage 
```
.
├── Data
│   ├── Curvilinear-Sierpinski-data.npy
│   ├── Henon-data.npy
│   └── Logistic-data.npy
│   └── Sierpinski-data.npy
├── README.md
├── examples.py
├── requirements.txt
├── utils.py
```
